Shopping Cart System

A full-stack Shopping Cart Management System built using React (frontend), Flask (backend), and Oracle Database with PL/SQL stored procedures.

This project supports complete e-commerce cart functionality including user registration, login, cart management, order placement, payment recording, stock management, invoice generation, and admin sales dashboard.

Features 

User Features :

User Registration
User Login Authentication
View Products
Add Items to Cart
Update Cart Quantity
Remove Items from Cart
View Cart
View Cart Total
Place Order
View Order History
Generate Invoice
Payment Processing

Inventory Features :

Real-time Stock Validation
Automatic Stock Reduction after Checkout
Prevents Over-ordering
Out-of-stock handling

Admin Features:

Sales Dashboard
Revenue Tracking
Product-wise Sales Analysis

Database Features:

Oracle SQL database integration
PL/SQL Stored Procedure for order placement
Transaction-safe stock updates
Relational schema design
